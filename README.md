# Schedule Builder Streamlit Version
Version 1 of my Streamlit Compliant Schedule Builder

This takes five files loaded with your data, and gives you a schedule with a percentage completion statistic.  Tries to make teacher schedules as well.

Input:
Course Data
Student Requests
Teacher Data
Disallowed Course Pairs
Disallowed Periods

Output:
Course Enrollment Summary
Master Schedule Grid
Student Master Schedule List by Student
Unscheduled Students
Teacher Schedules
